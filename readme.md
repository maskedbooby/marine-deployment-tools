# Marine Deployment Tools

Automation scripts used for deploying and managing maritime telemetry services.

## Overview

This repository contains internal tooling used for:

- Container-based service deployment
- Monitoring setup for tide and telemetry systems
- Infrastructure automation across production environments

## Environment

- Region: ap-south-1
- Deployment: Docker-based services
- Logging: centralized marine monitoring logs

## Components

- deploy.sh → handles container deployment
- config.py → service configuration
- environment variables managed during runtime

## Notes

Temporary debugging artifacts may be created during deployment testing and should be removed before production pushes.
